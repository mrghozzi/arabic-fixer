# Arabic Encoding Fixer

إضافة لإصلاح النصوص العربية المعطوبة الناتجة عن مشاكل الترميز `Mojibake` بعد استيراد قاعدة بيانات قديمة بترميز غير صحيح.

## المزايا
- واجهة إدارة حديثة داخل `/admin/arabic-fixer`.
- معاينة منظمة حسب الجدول قبل أي تعديل.
- نطاقات فحص جاهزة مع إمكانية اختيار الجداول يدوياً.
- فصل إصلاح النصوص عن تحويل `charset/collation`.
- تقارير أوضح للنتائج والتحذيرات والأخطاء.

## خطوات الاستخدام
1. افتح صفحة `/admin/arabic-fixer`.
2. اختر نطاق الفحص أو الجداول المطلوبة.
3. شغّل المعاينة وراجع النتائج.
4. أكّد وجود نسخة احتياطية.
5. نفّذ الإصلاح.

## ملاحظات
- يفضّل أخذ نسخة احتياطية كاملة من قاعدة البيانات قبل التشغيل.
- خيار تحويل الترميز اختياري ويُطبّق فقط على الجداول المحددة.
- إذا لم تكن البيئة MySQL/MariaDB فسيتم تجاهل خيار تحويل الترميز تلقائياً.
